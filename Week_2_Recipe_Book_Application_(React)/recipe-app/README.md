# 🍳 Recipe Book Application

A responsive React application that fetches, searches, filters recipes from the DummyJSON API.

---

## Tech Stack
* **Frontend**: React
* **Styling**: CSS (Tailwind)
* **Data Source**: DummyJSON Recipes API
* **Version Control**: Git & GitHub

---

## Features
* **Recipe Browsing**: Displays all recipes in a card grid format featuring tags, name, image, difficulty, cuisine, and cook time.
* **Detailed View**: Click any recipe to view comprehensive details including serving sizes, ingredients, and step-by-step instructions.
* **Live Search**: Real-time search functionality to find recipes by title.
* **Difficulty Filtering**: Filter recipe lists dynamically by difficulty level (Easy, Medium, Hard).
* **Dynamic Loading**: Load recipes incrementally using limit and skip parameters.
* **Error Handling**: Graceful error catching with informative user feedback during network or API failures.
* **Responsive Layout**: Fully adaptive design optimized for mobile, tablet, and desktop screens.
